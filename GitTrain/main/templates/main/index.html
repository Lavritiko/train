Это небольшой тренажер по Git. Выполни задания и не забудь что-нибудь добавить на страницу почета :)

Если появилось желание запустить сервер и посмортеть как сейчас выглядит сайтик, могу дать несколько советов.

1. В командной строке перейди в папку с проектом

2. Затем введи venv\Scripts\activate для запуска окружения

3. Войдя в окружение, смело вводи python manage.py runserver для запуска сервера

4. Перейди по адресу 127.0.0.1:8000 и наслаждайся :)
